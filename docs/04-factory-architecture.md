# Factory Architecture

## Overview

The factory is a square, modern concrete building viewed in **side-on dollhouse cutaway** — one entire wall removed to reveal all floors simultaneously. It grows from 1 floor to 4 floors across the narrative arc.

Each floor has:
- A distinct **purpose** and **visual identity**
- Specific **Meerklet roles** assigned to it
- A unique **ambient color** derived from the brand palette
- Characteristic **activity patterns** and **machinery**
- **Vertical connections** to adjacent floors (staircases, data conduits, elevators)

---

## Floor 1 — Infrastructure Layer (Ground Floor)

**Theme**: The engine room. Foundational. Always running.
**Ambient Glow**: Cobalt blue (`#2843d0`) with green (`#08926d`) status indicators
**When it appears**: Act 2 (built as foundation when factory construction begins)

### Visual Elements

| Element | Description | Animation |
|---------|-------------|-----------|
| **Server racks** | Rows of blinking server cabinets with status LEDs | LEDs blink in patterns, occasional component swap by Meerklet |
| **Data pipeline tubes** | Translucent tubes carrying glowing data packets | Packets (small glowing cubes) flow continuously through tubes |
| **Cooling systems** | Ventilation grilles with neon-lit fans | Fans spin, cool mist occasionally visible |
| **Network nodes** | Hexagonal routing stations where pipes converge | Data packets split, merge, and redirect at nodes |
| **Cloud platforms** | Floating holographic platforms above the server racks | Gentle bobbing motion, connected by light beams to racks |
| **Power conduits** | Glowing cable runs going up through the ceiling to higher floors | Steady pulse of energy flowing upward |
| **Monitoring station** | Wall of small screens showing system health | Screens refresh with live metrics |

### Meerklet Presence
- **Deployers** (orange vests — Infrastructure): Operating pipeline controls, pushing deployment packages
- **Monitors** (orange vests — Infrastructure): Watching dashboard screens, adjusting configurations

### Activity Pattern
Steady, rhythmic, reliable. This floor never stops. The pace is consistent — it's the heartbeat of the factory.

### Sound Design Notes
- Low hum of servers
- Gentle whoosh of cooling systems
- Soft beeps from monitoring stations
- Data packet "chimes" as they flow through tubes

## Floor 2 — Agent Factory Floor

**Theme**: The production line. Where Meerklets are built, trained, and deployed.
**Ambient Glow**: Purple (`#b984ff`) training chambers, cobalt blue (`#2843d0`) assembly stations
**When it appears**: Act 1 (this is the original floor where everything starts)

### Visual Elements

| Element | Description | Animation |
|---------|-------------|-----------|
| **Assembly line** | Central conveyor belt with robotic arms on each side | Arms swing, pick, place, weld. Sparks fly |
| **Robotic arms** | Articulated arms that assemble Meerklet bodies from components | Precise, mechanical movements. Different arms for different parts |
| **Training chambers** | Glass pods where new Meerklets sit and absorb knowledge | Swirling data streams circle the Meerklet inside. LED pulses |
| **Quality gates** | Checkpoint stations with green/red indicators | Meerklets walk through, gate scans, green light = pass |
| **Component racks** | Shelving with Meerklet parts — heads, bodies, vests, badges | Occasionally a Meerklet or arm picks a component |
| **Conveyor belts** | Multiple belts carrying code modules, tools, and components | Items move along belts, sometimes redirected at junctions |
| **Configuration stations** | Desks with screens showing agent specs and parameters | Meerklets being configured before deployment |
| **Deployment chute** | A slide/elevator connecting Floor 2 to Floor 1 | Completed Meerklets slide down to their assigned floor |

### Meerklet Presence
- **Builders** (yellow vests `#e6b30d` — Production): Assembling new Meerklets, constructing code modules
- **Testers** (yellow vests `#e6b30d` — Production): Running quality checks, inspecting outputs

### Activity Pattern
Dynamic, industrial, energetic. This is the busiest floor visually. Constant motion, sparks, assembly, testing.

### The Assembly Sequence (Key Animation)
1. Robotic arm picks a blank body shell from the rack
2. Another arm installs the LED chest stripe
3. A third arm places the head (with glasses)
4. The Meerklet is placed in a training chamber — data swirls around it
5. It emerges, walks through the quality gate
6. A Meerklet at the configuration station assigns it a role and vest color
7. The new Meerklet puts on its vest and heads to its assigned floor

## Floor 3 — Orchestration Layer

**Theme**: The control room. Where workflows are designed and agent activity is coordinated.
**Ambient Glow**: Purple (`#6326b3`) with holographic blue-white overlays
**When it appears**: Act 4

### Visual Elements

| Element | Description | Animation |
|---------|-------------|-----------|
| **Central holographic display** | Large 3D node graph showing active agent workflows | Nodes pulse, connections glow, workflow paths animate |
| **Workflow design stations** | Desks where orchestration patterns are configured | Meerklets drag-and-drop workflow nodes |
| **Dashboard wall** | Wall-mounted screens showing throughput, success rates, errors | Real-time updating numbers and charts |
| **Communication arrays** | Antenna/dish structures for agent-to-agent coordination | Subtle radio wave animations between arrays |
| **Pipeline vis** | Glowing tubes showing work items from request to completion | Work items (small glowing orbs) flow through the pipeline |
| **Kanban board** | Holographic board with To Do / In Progress / Done columns | Cards move across columns, some glow green (done), amber (in progress) |
| **Escalation terminal** | Station where issues are flagged up to Floor 4 | Red alert pulses when something needs human attention |

### Meerklet Presence
- **Architects** (magenta vests — Orchestration): Designing workflows, configuring orchestration patterns
- **Reviewers** (magenta vests — Orchestration): Reviewing outputs, approving releases, flagging issues

### Activity Pattern
Cerebral but busy. Less physical motion than Floor 2, more data motion. Screens updating, holograms shifting, workflow graphs reconfiguring.

## Floor 4 — Human Command Centre (Top Floor)

**Theme**: Strategic command. Where humans steer the factory.
**Ambient Glow**: Warm white with cobalt accent lighting. Greenery adds organic contrast.
**When it appears**: Act 5

### Visual Elements

| Element | Description | Animation |
|---------|-------------|-----------|
| **Panoramic windows** | Full-width windows overlooking the business park | Subtle exterior lighting changes, clouds drift |
| **Aleksandr's command chair** | Elevated chair with wraparound holographic displays | Displays cycle through factory KPIs |
| **Sergei's workbench** | Engineering station with tools, diagnostics, Floor 3 link | Sergei fine-tunes, adjusts, occasionally responds to alerts |
| **Oleg's learning station** | Interactive table with Meerklets and experiments | Oleg plays/experiments, Meerklets respond |
| **Strategy table** | Large holographic surface for roadmaps and planning | 3D roadmap with milestones, timelines |
| **Communication hub** | Screens showing requests in, features out | Requests flow in from left, completed features flow out right |
| **Living wall** | Vertical garden integrated into the back wall | Subtle growth animation, contrast with tech |
| **Stairwell access** | Visible stairs/elevator down to Floor 3 | Occasional Meerklet comes up with a report |

### Character Positions
- **Aleksandr**: Command chair, centre-left, elevated
- **Sergei**: Workbench, centre-right, working level
- **Oleg**: Learning station, right side, ground level

### Activity Pattern
Thoughtful, strategic, calm. The contrast with the busy floors below is deliberate — humans have moved from doing the work to steering the work.

## Vertical Connections

The floors are connected by visible infrastructure:

| Connection | Purpose | Visual |
|-----------|---------|--------|
| **Staircases** | Meerklet movement between floors | Open metal stairs on the cutaway side |
| **Data conduits** | Information flow between layers | Glowing cable bundles running vertically |
| **Deployment chute** | Sending completed Meerklets to their stations | Slide/tube from Floor 2 down to Floor 1 |
| **Escalation tube** | Flagging issues from Floor 3 up to Floor 4 | Pneumatic tube that sends glowing red capsules up |
| **Power lines** | Energy from Floor 1 to all floors | Pulsing blue lines in the walls |
| **Elevator** | Meerklet and cargo transport | Small goods elevator on the far side |

## Factory Growth Sequence

| Act | Floors Visible | Building Height | Chimney |
|-----|---------------|----------------|---------|
| Act 1 | 1 (just the workspace, becomes Floor 2) | Single storey | None |
| Act 2 | 2 (Floor 1 + Floor 2) | Double storey | Installed, small puffs |
| Act 3 | 2 (same, but obstacles visible) | Double storey | Intermittent, disrupted |
| Act 4 | 3 (Floors 1-3) | Triple storey | Steady stream |
| Act 5 | 4 (Floors 1-4) | Full height | Constant, impressive |
